<h2>Ellie Castelli Rental Division</h2>
Platform where real-estate owners can list their properties for rent.<br /><br />

<b>Build with:</b><br />
<li>Next.js 14</li>
<li>NextAuth.js</li>
<li>OAuth 2.0</li> 
<li>MongoDB</li> 
<li>Tailwind</li> 
<li>Shadcn/ui</li> 
<li>react-photoswipe-gallery</li>
<li>Cloudinary</li>
<li>opencage-api-client (geocoding)</li> 
<li>Leaflet OpenStreetMap</li><br />

<b>Features:</b><br/>

<ul><li>register</li>
<li>login with:</li>
<li>credentials</li>
<li>email</li>
<li>google</li>
<li>facebook</li></ul>
Upload property<br/>
Edit uploaded property<br/>
Search property<br/>
Add property<br/>
Send mail to property owner<br />
List messages <br/><br />
